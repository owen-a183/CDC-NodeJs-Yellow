# CDC-NodeJs-Yellow

Assignment for CDC- NodeJS Yellow belt (Todo App)

## Installation
Might just installing nodemon, really not using any module except http and fs
```bash
npm i
```

## Usage
Run with nodemon module
```bash
npm run server
```

Then, access your localhost
